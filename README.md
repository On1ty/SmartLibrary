## Deployment

To deploy this project run follow these:

```bash
  npm install
```

```bash
  ionic serve
```
or

```bash
  ionic serve -l
```

you may need to install this 

```bash
  npm i @ionic/lab
```
  



## Features

- Light/dark mode toggle
- Cross platform (IOS/Android)
- QR Code Scanning
- Facial Recognition (soon)
